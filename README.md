# YakoEngine

## 💻 Prerequisites (System Requirements)

To build this project, you only need to ensure that the core development environment is installed and available in your **PATH**:

1. **IDE & Compiler:** Visual Studio 2022 (with the *"Desktop development with C++"* workload and MSVC v193 toolset).
2. **Build System:** CMake (version **4.2.1** or higher).
3. **Interpreter:** Python with Pip.
4. **Package manager:** Conan.
5. **Formatting:** clang-format.

Go to Scripts/Your-platform/ and run any .bat file for build.
